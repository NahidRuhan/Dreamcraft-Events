###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: `{}`
 
<i>Greeting has an empty object inside of it. So console logging it will show output of an empty object. The typo in the variable name (greetign instead of greeting) will not affect the output. Output: {}</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: `"12"`

<i>Sum function is taking 2 input and adding them together and returning it. If we had sent 2 int as variable it would've summed it and return. Since one of our input is string, JavaScript performs string concatenation. The function sum takes 1 as a number and "2" as a string, so it concatenates them, resulting in "12."</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: `Hi there, undefined`

<i>Function sayHi receives a parameter 'name' and returns a string "Hi there, (name)". Since we didn't send any parameter, the "name" will remain undefined.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>We are looping through nums array and incrementing count by boolian if condition checking. Since the first index of the array is 0 which is false in boolian, count wont increment. But it will increment for rest of the array.</i>

</p>
</details>
